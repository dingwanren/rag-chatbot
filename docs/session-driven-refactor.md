## 🔒 安全限制

### ✅ 已实现

1. **禁止无 chatId 发送消息**
   ```typescript
   const handleSendMessage = async (messageText: string) => {
     if (!chatId) return  // ✅ 验证
     // ...
   }
   ```

2. **禁止首页自动创建聊天**
   - ❌ 删除了输入框
   - ❌ 删除了 `onSubmit` 自动创建逻辑
   - ✅ 必须通过 Modal 显式创建

3. **RAG 聊天必须有知识库**
   ```typescript
   if (chatInfo?.mode === 'rag' && !chatInfo.knowledge_base_id) {
     message.error('该聊天未绑定知识库，无法发送消息')
     return
   }
   ```

4. **一个聊天只能绑定一个知识库**
   - 创建时绑定
   - 不可修改

## 📊 数据流

```
用户打开应用
    ↓
默认首页 /
    ↓
显示空状态页 + "新建对话"按钮
    ↓
点击"新建对话"
    ↓
打开 CreateChatModal
    ↓
选择模式 + 知识库
    ↓
创建聊天 (chats 表)
    ↓
跳转 /chat/[chatId]
    ↓
显示聊天页面
    ↓
发送消息
    ↓
验证 chatId + 知识库
    ↓
调用 RAG API
    ↓
显示回答
```
