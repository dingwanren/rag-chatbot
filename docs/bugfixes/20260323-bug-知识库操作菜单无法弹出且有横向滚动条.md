# 知识库操作菜单无法弹出且有横向滚动条

## 基本信息

- **日期**: 2026-03-23
- **问题类型**: bug
- **严重等级**: P2-一般
- **涉及模块**: `app/components/knowledge-base-list.tsx`
- **相关 Issue**: 无

## 问题描述

### 现象

1. 点击知识库列表项的 "..." 操作按钮后，下拉菜单没有弹出
2. 页面底部出现横向滚动条

### 复现步骤

1. 进入知识库管理页面
2. 鼠标悬停在任意知识库列表项上
3. 点击右侧出现的 "..." 按钮
4. 观察菜单是否正确弹出
5. 检查页面底部是否出现横向滚动条

### 错误信息

无控制台错误，纯 UI 交互问题

## 根因分析

**关键代码位置**: `app/components/knowledge-base-list.tsx:190-230`

### 问题 1：菜单无法弹出

**原因**: 使用 DOM 操作 (`document.getElementById`) 控制菜单显示/隐藏，但菜单位置计算错误：

```javascript
// 问题代码
const rect = e.currentTarget.getBoundingClientRect()
const menu = document.getElementById(`menu-${kb.id}`)
if (menu) {
  menu.style.top = `${rect.bottom + 4}px`
  menu.style.left = `${rect.left}px`  // 定位到左侧，可能超出容器
  menu.classList.toggle('hidden')
}
```

- 菜单使用 `absolute` 定位，但父容器没有正确的 `relative` 上下文
- 使用 `left` 定位可能导致菜单超出左侧边界
- 依赖 DOM 操作而非 React state，不符合 React 最佳实践

### 问题 2：横向滚动条

**原因**: 

1. 菜单使用 `absolute right-0` 定位，但父容器宽度固定
2. 菜单最小宽度 `min-w-[120px]` 可能超出可用空间
3. 菜单位置计算时使用 `left` 值，导致菜单向右超出容器边界

## 修复方案

### 方案描述

1. 使用 React state (`openMenuId`) 控制菜单显示/隐藏
2. 修正菜单位置为 `right-0 top-full mt-1`，相对于父容器右下角定位
3. 添加点击遮罩层关闭菜单的功能
4. 移除所有 DOM 操作代码

### 关键代码变更

**修改前**:
```tsx
const [inputValue, setInputValue] = useState('')
// 无 openMenuId state

// DOM 操作控制菜单

  onClick={(e) => {

    const rect = e.currentTarget.getBoundingClientRect()

      menu.style.top = `${rect.bottom + 4}px`
      menu.style.left = `${rect.left}px`
      menu.classList.toggle('hidden')
    }
  }}
>
  ...
</button>
<div
  id={`menu-${kb.id}`}
  className="hidden absolute right-0 z-10 ..."
>
  ...
</div>
```

**修改后**:
```tsx
const [inputValue, setInputValue] = useState('')
const [openMenuId, setOpenMenuId] = useState<string | null>(null)

// React state 控制菜单
<button
  onClick={(e) => {
    e.stopPropagation()
    setOpenMenuId(openMenuId === kb.id ? null : kb.id)
  }}
  className="... opacity-0 group-hover:opacity-100 ..."
>
  ...
</button>
{openMenuId === kb.id && (
  <>
    <div className="fixed inset-0 z-10" onClick={() => setOpenMenuId(null)} />
    <div className="absolute right-0 top-full mt-1 z-20 ...">
      <button onClick={() => { setOpenMenuId(null); handleOpenRenameModal(kb) }}>
        重命名
      </button>
      <button onClick={() => { setOpenMenuId(null); handleDelete(kb) }}>
        删除
      </button>
    </div>
  </>
)}
```
